# Hitbus
### Global Startup Demand Intelligence Platform

Hitbus helps founders validate startup ideas before they build — using verified, structured demand signals instead of hype or opinions.

Built with: Python · FastAPI · MySQL

## Status
🚧 Actively building in public

## Architecture
- Two-sided platform: Founders + Validators as distinct roles
- Weighted demand scoring (not simple vote counts)
- Identity verification required — no anonymous users
- Anti-manipulation logic built into the data layer

## Stack
- Backend: Python + FastAPI
- Database: MySQL
- Auth: bcrypt + JWT (in progress)
